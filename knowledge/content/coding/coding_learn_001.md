# 哈希表基础与应用

哈希表是一种通过键值对存储数据的数据结构，能够在O(1)时间复杂度内完成查找、插入和删除操作。掌握哈希表是解决「两数之和」等经典问题的关键。

## 核心要点

- 哈希表的核心思想：用空间换时间
- Python中dict就是哈希表的实现
- 常见应用：快速查找、去重、计数
- 哈希冲突的处理方法

## 示例

```python
# 使用哈希表记录出现过的数
seen = {}
for num in nums:
    if target - num in seen:
        return [seen[target-num], i]
    seen[num] = i
```

- 遍历数组时，用哈希表记录每个数及其索引，查找配对数只需O(1)时间
